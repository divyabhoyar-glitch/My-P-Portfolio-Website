## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/divyabhoyar-glitch/My-P-Portfolio-Website.git
```

2. Open the project folder

```bash
cd My-P-Portfolio-Website
```

3. Open `index.html` in your browser.
